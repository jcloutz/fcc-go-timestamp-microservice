Timestamp Microservice

This is a timestamp microservice written with Go as a solution to Free Code Camp's Timestamp Microservice API Project

View Source

Instructions

  • Unix Timestamp: Pass a unix timestamp. /1231231231
  • Date String: Pass a date string in the below format(s) /December 15, 2015
    /December%2015,%202015
  • Expected Return Value: The return will be a json object containing a unix timestamp and a natural human readable date. 
    {
    "unix": 1450137600,
    "natural": "2015-12-15 00:00:00 +0000 UTC"
}
